# MindPulse  
## Information  
**Project type:** Prototype  
**Project description:** A booking system for individuals seeking mental health services  
**Owner:** Lee Ming Jun, Ryan Chan  
  
## Tech Stack  
**Frontend:** React.js  
**Backend:** Java Spring Boot  
**Database:** AWS DynamoDB  
**Storage:** AWS S3  
**Compute:** AWS EC2
